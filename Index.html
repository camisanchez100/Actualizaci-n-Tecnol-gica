HTML Introduction

What´s HTML?

  • Es un estandar para el Hypertext(hipertexto) del maquetado, es un estandar para representar la estructura para colocar la información
  • Escribe la estructura de la página web
  • Consiste en una serie de elementos
  • Los elementos le dicen al navegador como mostrar el contenido
  • Los elementos tambien consisten en piezas de etiquetas con contenido

Elementos Basicos Explicados

  1. La etiqueta "DOCTYPE" define la declaración del documento como HTML
  2. La etiqueta HTML es el elemento principal de una página HTML
  3. La etiqueta "head" contiene información adicional acerca de la página HTML
  4. La etiqueta "title" especifica el titulo de la página web
  5. La etiqueta "body" especifica el cuerpo de la página web (todos los elemtos vacios)
  6. La etiqueta "h1" se define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño subtitular
  8. La etiqueta "p" define un texto de tamaño parrafo

¿Qué es un elemento en html?

Un elemnto html esta definido por una etiqueta de inicio seguido por un contenido y una etiqueta de finalización
  • <nombreEtiqueta> etiqueta de inicio
  • <Contenido> entre la etiqueta de inicio y la etiqueta de finalizacion
  • <nombreEtiqueta> etiqueta de finalizacion

    Etiqueta <map>

    La etiqueta <map> define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distintas areas detectadas por el navegador, logrando identificar de manera independiente cada una de las areas. La etiqueta <map> define las areas de la imagen mediante la etiqueta <area> con los atributos HTML "shape" que define la forma del area señalada en la imagen, "coords" que define la coordenadas de cada uno de los puntos de la shape, "href" que puede definir un hipervinculo para cada una de las areas. Ejemplo:
    • <map src = "workflow.jpg" usemap = "#workmap">
    • < map name = "workmap">
    • < area shape = "rect" coord = "34,44,270,350" href = "Pagina1.com" >
    • < area shape = "cicle" coord = "200,172,333,250" href= "pagina2.com">
